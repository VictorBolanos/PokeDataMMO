## 📖 Overview

PokeDataMMO is a modern, full-featured web application designed to enhance your PokeMMO journey. From managing berry farms to building competitive PVP teams, this tool provides everything you need in a beautiful, user-friendly interface.

### 🎯 Why PokeDataMMO?

- ✅ **All-in-One Platform** - Multiple tools unified in one place
- ✅ **Cloud Sync** - Save your data with Firebase integration
- ✅ **Beautiful UI** - Modern, responsive design with dark/light themes
- ✅ **Multi-language** - Full support for English and Spanish
- ✅ **Real-time Calculations** - Instant results with auto-save
- ✅ **Free & Open Source** - No paywalls, no ads

## ✨ Features

### 🌱 **Berry Farming Calculator**
> **Status:** ✅ **FULLY IMPLEMENTED**

Advanced calculation system to optimize berry cultivation and maximize profits.

**Key Features:**
- 📅 **Irrigation Schedules** - Automatic calculation of planting, irrigation, and harvest times
- 💰 **Profit Analysis** - Real-time calculation of seed sales and extraction costs
- 📊 **Expense Tracking** - Manage purchase and management costs
- 💾 **Cloud Save** - Store unlimited farming calculations in Firebase
- 🔄 **Auto-save** - Changes saved automatically every 10 seconds
- 🌍 **Multi-language** - Dynamic ES/EN translation

**How it works:**
1. Create or load a saved calculation
2. Select cultivation type (Leppa/Zanamas)
3. Input plants per character and number of characters
4. Set irrigation schedules and harvest data
5. Track profits and expenses automatically
6. Everything saves to the cloud automatically

---

### ⚔️ **PVP Teams Manager**
> **Status:** ✅ **FULLY IMPLEMENTED**

Complete team building and management system for competitive play.

**Key Features:**
- 🎯 **6-Pokémon Teams** - Build full competitive teams
- 📊 **Stats Calculator** - Automatic calculation with EVs, IVs, and Nature
- 🔢 **EV/IV Tracking** - Visual validation (max 510 EVs, 31 IVs)
- 🧬 **Nature System** - PokeAPI integration with stat modifications (+10%/-10%)
- ⚡ **Abilities & Moves** - Select from available options per Pokémon
- 🎒 **Items** - Equip items to your team members
- 💾 **Cloud Teams** - Save unlimited teams to Firebase
- 🔄 **Auto-save** - Teams saved automatically every 10 seconds

**How it works:**
1. Create new team or load existing one
2. Click [+] on any of 6 slots to add Pokémon
3. Search by name or ID
4. Configure EVs, IVs, Nature, Ability, Item, and Moves
5. Stats calculated automatically using official formulas
6. Team auto-saves to Firebase

## 🚀 Installation

### **Option 1: Use Online (Recommended)**
Simply visit: [https://victorbolanos.github.io/PokeDataMMO](https://victorbolanos.github.io/PokeDataMMO)

### **Option 2: Run Locally**

```bash
# Clone the repository
git clone https://github.com/VictorBolanos/PokeDataMMO.git

# Navigate to project directory
cd PokeDataMMO

# Open with a local server (required for modules)
# Using Python:
python -m http.server 8000

# Using Node.js:
npx http-server

# Visit: http://localhost:8000
```

## 📚 User Guide

### **Getting Started**

1. **Create an Account**
   - Click on user icon in top right
   - Register with username and password
   - Email is optional

2. **Navigate the Tools**
   - Use the tab system to switch between features
   - Each tool has its own dedicated interface

3. **Customize Your Experience**
   - 🎨 Change color theme
   - 🌙 Toggle dark/light mode
   - 🔤 Select your preferred font
   - 🖼️ Choose a Pokémon wallpaper
   - 🎵 Play PokeMMO music

### **Berry Farming Calculator**

```
1. Go to "Berry Farming" tab
2. Click "New Calculation" or load existing one
3. Enter calculation name
4. Select cultivation type (Leppa/Zanamas)
5. Set plants per character and number of characters
6. Fill irrigation schedules
7. Input harvest data and prices
8. Add purchase/management costs
9. View real-time profit calculations
10. Everything auto-saves to cloud ✅
```

### **PVP Teams Manager**

```
1. Go to "PVP Teams" tab
2. Click "New Team" or load existing one
3. Enter team name
4. Click [+] to add Pokémon (up to 6)
5. Search and select Pokémon
6. Configure:
   - EVs (0-252 per stat, max 510 total)
   - IVs (0-31 per stat)
   - Nature (affects +10%/-10%)
   - Ability
   - Held Item
   - 4 Moves
7. Stats calculate automatically
8. Team auto-saves to cloud ✅
```
